# ♟ Тульская Шахматная Гостиная

Видеоконференц-платформа для Тульского шахматного клуба.  
Аналог Google Meet с демонстрацией экрана, чатом, эмодзи, поднятием руки и правами администратора.

---

## 📋 Функциональность

- 🎥 **Видео и аудио** — WebRTC, HD-качество
- 🖥 **Демонстрация экрана** — полный экран или окно
- 💬 **Чат** — в реальном времени с эмодзи
- ✋ **Поднять руку** — уведомление всех участников
- 😄 **Эмодзи-реакции** — анимированные всплывающие реакции
- 🔇 **Мут / видео** — горячие клавиши M, V, S, H
- 👑 **Администратор** — управление участниками, удаление из зала
- 🔗 **Безопасные ссылки** — криптографически сгенерированные токены

## 🚀 ДЕПЛОЙ НА RENDER (БЕСПЛАТНЫЙ ТАРИФ)

### Шаг 1 — Подготовьте репозиторий

1. Создайте аккаунт на [GitHub](https://github.com)
2. Создайте **новый репозиторий** (например: `chess-lounge`)
3. Загрузите все файлы проекта в репозиторий:
   ```
   chess-lounge/
   ├── server.js
   ├── package.json
   ├── public/
   │   ├── index.html
   │   ├── room.html
   │   ├── css/style.css
   │   └── js/
   │       ├── app.js
   │       └── room.js
   └── README.md
   ```

   Через терминал:
   ```bash
   cd chess-lounge
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/ВАШ_ЛОГИН/chess-lounge.git
   git push -u origin main
   ```

---

### Шаг 2 — Создайте сервис на Render

1. Перейдите на [render.com](https://render.com) и зарегистрируйтесь
2. Нажмите **"New +"** → **"Web Service"**
3. Выберите **"Connect a repository"** → авторизуйте GitHub и выберите `chess-lounge`
4. Заполните настройки:

   | Поле | Значение |
   |------|----------|
   | **Name** | `chess-lounge` (или любое) |
   | **Region** | Frankfurt (EU Central) — ближайший к России |
   | **Branch** | `main` |
   | **Runtime** | `Node` |
   | **Build Command** | `npm install` |
   | **Start Command** | `npm start` |
   | **Instance Type** | `Free` |

5. Нажмите **"Create Web Service"**

> ⏳ Первый деплой занимает 2–5 минут. Ваш сайт будет доступен по адресу вида:  
> `https://chess-lounge.onrender.com`

---

### Шаг 3 — Переменные окружения (опционально)

В разделе **Environment** → **Environment Variables** можно добавить:

| Key | Value | Описание |
|-----|-------|----------|
| `PORT` | `3000` | Порт (Render устанавливает автоматически) |
| `NODE_ENV` | `production` | Режим production |

---

### Шаг 4 — Настройка UptimeRobot (ОБЯЗАТЕЛЬНО для бесплатного тарифа!)

> ⚠️ **Важно:** На бесплатном тарифе Render сервер **засыпает** через 15 минут неактивности.  
> UptimeRobot будет "будить" его каждые 5 минут.

1. Перейдите на [uptimerobot.com](https://uptimerobot.com) и зарегистрируйтесь (бесплатно)
2. Нажмите **"Add New Monitor"**
3. Заполните:

   | Поле | Значение |
   |------|----------|
   | **Monitor Type** | `HTTP(s)` |
   | **Friendly Name** | `Chess Lounge` |
   | **URL** | `https://chess-lounge.onrender.com/ping` |
   | **Monitoring Interval** | `5 minutes` |

4. Нажмите **"Create Monitor"**

✅ Теперь сервер всегда активен во время встреч!

---

### Шаг 5 — Проверьте работу

1. Откройте `https://chess-lounge.onrender.com`
2. Введите ваше имя в форму **"Создать конференцию"**
3. Нажмите **"Открыть зал"**
4. Скопируйте ссылку для участников и отправьте им
5. Нажмите **"Войти в зал"** — вы попадёте в комнату как администратор

## 🔄 Обновление сайта

Чтобы обновить сайт после внесения изменений:

```bash
git add .
git commit -m "Обновление"
git push
```

Render автоматически задеплоит новую версию через 1–2 минуты.

## ⌨️ Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `M` | Вкл/выкл микрофон |
| `V` | Вкл/выкл камеру |
| `S` | Начать/остановить демонстрацию экрана |
| `H` | Поднять/опустить руку |

## 🏗 Технологии

- **Backend:** Node.js + Express + Socket.io
- **Frontend:** Vanilla JS + WebRTC API
- **Сигналинг:** Socket.io WebSockets
- **Шрифты:** Cormorant Garamond + Golos Text
- **Деплой:** Render.com
